## 💻 Projeto

Por meio desse projeto, pude treinar todo o conhecimento aquirido até então na trilha Explorer, e pude adicionar outras coisas que já havia estudado, como maneiras de deixar o site mais responsivo.

- Semântica correta;
- Responsividade (apliquei algumas técnicas que conhecia);
- Tags HTML;
- Flexbox e posicionamentos de elementos na tela.
